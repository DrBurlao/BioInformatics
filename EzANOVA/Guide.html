Welcome to EzANOVA

EzANOVA is a simple tool for conducting one-way ANOVA (Analysis of Variance) calculations. Follow these steps to perform an ANOVA analysis:

How to Use:

  1. Enter Data: Input the data for each group in the provided text fields. Separate values within each group by commas.
  2. Add or Remove Groups: Use the "Add Group" and "Remove Group" buttons to manage the number of groups for your analysis.
  3. Calculate ANOVA: Click on the "Calculate ANOVA" button to perform the ANOVA analysis based on the entered data.
  4. View Results: The ANOVA results, including F-statistic, p-value, and group means, will be displayed along with a bar chart representing the group means.
  5. Export Data: Optionally, you can export the ANOVA results to a CSV file by clicking the "Export to CSV" button.
