Ez Normalizer

This application allows laboratory professionals to normalize sample results for accurate data analysis. With intuitive controls, statistical calculations, and chart visualization, it's a valuable tool for laboratory and research work. It also supports CSV export, making it easy to save and share results.

Open Application

Key Features

  • Normalization: Calculate normalized results using the z-score method for accurate comparison.
  • Data Management: Add, remove, and clear sample data with intuitive controls.
  • Chart Visualization: Visualize original and normalized results with dynamic bar charts.
  • CSV Export: Export sample results as a CSV file for easy data sharing and record-keeping.
  • Statistics: Calculate and display statistical data such as mean, variance, and standard deviation.

Technical Requirements

  • Compatibility: Works with modern web browsers like Chrome, Firefox, and Safari.
  • Internet Connection: Required for loading external libraries like Chart.js and Font Awesome.

Security Considerations

This application does not store user data persistently. Users are responsible for securely saving any exported files. It does not collect or share personal information without explicit consent.

External Libraries:
  • Chart.js for data visualization (https://www.chartjs.org/)
  • Font Awesome for icons (https://fontawesome.com/)
  • Google Fonts for typography (https://fonts.google.com/)
