Welcome to EzNormalizer

EzNormalizer is a tool for normalizing experimental results using Z-scores. Follow these steps to use EzNormalizer:

How to Use:

  1. Enter Data: Input the experiment name, sample name, and result in the provided input fields.
  2. Load Data from CSV: Optionally, you can upload experimental data from a CSV file. The file should be formatted with columns for experiment, sample, and result.
  3. Clear Data: Use the "Clear All Data" button to remove all entered data.
  4. Export Normalized Data: Click on the "Export as CSV" button to download the normalized experimental results as a CSV file.
